# 项目管理(Project Management)

## 对项目的管理称作项目管理

把知识, 技能, 工具, 技术等各种各样的东西应用于[[项目(Project)]], 来满足项目的要求, 就叫项目管理

## 从目的对象上看项目管理

### 对人管理

- 人力资源
- 沟通
- 采购
- 相关人员

### 对事管理

- 范围
- 时间
- 成本
- 质量
- 风险

## 项目管理的周期

1. 项目启动者发起项目
2. 项目输入
3. 启动-规划-执行-循环-收尾
4. 交付成果+记录过程资产

# 重要三角函数不等式

![[重要三角函数不等式-1742369219925.jpeg]]

$S_{\triangle OBA} < S_{扇形 OBA} < S_{\triangle OCA}$ 
$S_{\triangle OBA} = \frac{1}{2} sin(x)$ 
$S_{扇形 OBA} = \frac{1}{2} x$ $S_{\triangle OCA} = \frac{1}{2} tan(x)$

# 树的基本结构

树的构成通常包括 **节点(node)** 和 **边(edge)** , 节点内包含一些数据, 以及指向子节点的引用
树通常有一个 **根节点(root)**

``` java
class TreeNode {
	int value;
	TreeNode childNode;
	
	TreeNode(int value) {
		this.value = value;
		this.childNode = childNode;
	}
}
```

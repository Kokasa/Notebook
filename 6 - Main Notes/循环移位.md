# 循环移位

```
对于不带CF的循环移位, 相当于在机器数旁边的寄存器记录刚刚移动的位

对于带CF的循环移位, 寄存器中也有一个初始值, 这个初始值一起参与移位
```

可以简单的将循环移位过程看作是一个环结构
不带CF位的循环移位, 操作数本身每个位的数循环, 寄存器用来记录当前发生改变的位
带CF位的循环移位, 这个寄存器本身也在这个环结构中, 不仅记录当前发生改变的位, 其本身的初始数据也会沿着环结构前进
